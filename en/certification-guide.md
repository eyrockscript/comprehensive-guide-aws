## Cloud Practitioner (Foundational)

### Description
Entry-level certification demonstrating understanding of AWS Cloud fundamentals.

### Target Audience
- Sales professionals
- Technical marketers
- Project managers
- Executives
- Candidates for technical roles

### Exam Content
| Domain | Weight |
|--------|--------|
| Cloud Concepts | 24% |
| Security and Compliance | 30% |
| Technology | 30% |
| Billing and Pricing | 16% |

### Study Resources
- **Official course**: AWS Cloud Practitioner Essentials (6 hours)
- **Practice**: Cloud Quest
- **Whitepapers**: Overview of AWS, AWS Well-Architected Framework
- **Exam cost**: $100 USD

### Tips
- Focus on concepts, not technical implementation
- Understand the shared responsibility model
- Master main services by name and function

## Effective Preparation

### Suggested Study Schedule

| Certification | Study Hours | Previous Experience |
|---------------|-------------|---------------------|
| Cloud Practitioner | 20-30 | 0-6 months |
| Solutions Architect Associate | 60-80 | 6-12 months |
| Developer Associate | 50-70 | 6-12 months |
| SysOps Administrator | 70-90 | 12+ months |
| Solutions Architect Professional | 80-120 | 2+ years |
| DevOps Engineer Professional | 80-120 | 2+ years |
| Specialty | 60-100 | Varies |

### Recommended Resources

**Official Courses**:
- AWS Skill Builder (free tier available)
- AWS Classroom Training
- AWS Workshops (hands-on)

**Practice**:
- AWS Free Tier
- Qwiklabs
- CloudAcademy
- A Cloud Guru

**Documentation**:
- AWS Whitepapers
- AWS FAQs per service
- AWS Architecture Center

**Practice Exams**:
- Official AWS Practice Exams
- Whizlabs
- Tutorials Dojo

### Exam Tips

1. **Read carefully**: Many questions have clues in the statement
2. **Eliminate obvious answers**: Discard clearly incorrect options
3. **Prioritize**: "More secure" vs "Cheaper" depending on context
4. **Managed services**: Prefer managed services over self-managed
5. **Multi-AZ**: Always consider high availability
6. **Cost optimization**: Don't forget cost factor in solutions

### After the Exam

**Digital Badge**:
- Share on LinkedIn
- Add to email signature
- Update your resume

**Recertification**:
- Associate: 3 years
- Professional: 3 years
- Specialty: 3 years

**Next steps**:
- Apply knowledge to real projects
- Consider next certification
- Share your experience with the community
